GETTING STARTED

Install & Launch in 3 Simple Steps

Get your local workspace up and running instantly. Run hierarchical agent networks on your machine with full state control.

STEP 01 download_for_offline

Run the Setup Script

Initialize the Python virtual environment, install agent dependencies, and configure local Redis daemon components.

$ ./setup.sh
STEP 02 key

Configure Credentials

Add your LLM provider credentials and Redis connection details in the Settings pane or paste directly into `.env`.

# Edit langneurons/.env
STEP 03 rocket_launch

Boot the Console

Fire up the FastAPI console server and start orchestrating your multi-agent team inside the interactive graph dashboard.

$ ./run.sh

Simple Initialization

Deploy complex swarms with a few lines of Python.

article langneurons.py 
from langneurons import Neuron

# Define a specialized neuron with specific constraints
swarm = Neuron(
    role="coordinator",
    tools=[search, code_exec],
    memory="isolated",
    hierarchy_level=1
)

# Activate the hierarchical cluster
swarm.activate()
