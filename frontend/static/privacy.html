PRIVACY OVER SOVEREIGNTY

Privacy Policy

Last updated: July 17, 2026. At LangNeurons, we believe that your multi-agent architecture and agent reasoning prompts represent your primary intellectual property. We design our software so you retain absolute ownership and control over your data.

dns

1. Zero Cloud Data Collection

The LangNeurons framework operates entirely within your designated host environment. We do not maintain any centralized telemetry reporting, diagnostic feedback loop, or external trace logging servers. Any telemetry streams generated during execution are visual-only or recorded in your local Redis context database.

encrypted

2. Local Credentials Isolation

Your API keys (Google Gemini, OpenAI, Moonshot, or OpenRouter) and Redis connection parameters are saved locally in the `langneurons/.env` file on your machine. The FastAPI backend only reads these configurations to authenticate directly with your model endpoints over encrypted Secure Socket Layer (SSL/HTTPS) connections. Keys are never transmitted, cached, or visible to LangNeurons Inc.

history

3. Thread Memory & Persistence

Agent memory, conversation traces, and task delegation histories are written to your local Redis instance. If you run the setup script, Redis is locked to localhost/127.0.0.1. You have complete control to wipe, flush, or migrate this database at any point.

terminal

4. Subprocess Sandboxing

Swarms are run locally using Python subprocess forks within the working directories. Any temporary scripts created for session executions are deleted immediately upon process termination.
