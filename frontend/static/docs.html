LangNeurons Core Docs

LangNeurons is an open-source framework built to construct, compile, and execute hierarchical multi-agent teams. Instead of flat agent interactions, LangNeurons groups workers under designated supervisor nodes to establish strict reporting lines, enhancing performance on complex tasks.

Installation & Setup Guide

LangNeurons can be set up in under 60 seconds. Run the setup command from your terminal to install the virtual environment:

./setup.sh content_copy

Next, configure your local environment variable credentials. Make sure to define your API keys and models in `langneurons/.env`:

# Model Keys

GEMINI_API_KEY=AIzaSy...

OPENAI_API_KEY=sk-proj-...

MOONSHOT_API_KEY=sk-kimi-...

# Orchestration parameters

ROUTER_MODEL=gpt-4o

THINKING_MODEL=gemini-1.5-pro

Once configuration parameters are stored, launch the web application server:

./run.sh content_copy

Swarm Benefits

Isolated State Context

Each agent operates in its own memory silo, preventing prompt pollution and scaling to dozens of tools cleanly.

Dynamic Execution Paths

The framework automatically routes queries and spawns temporary clean executors, eliminating process memory leaks.

Step-by-Step Guide

  1. Design the Hierarchy: Use the drag-and-drop React Flow dashboard to create agent nodes.
  2. Link Capabilities: Draw connection edges from the supervisor node to workers to represent authority chains.
  3. Compile Node Code: Open the "Swarm Code" drawer, review generated script files, and test using the Chat interface.
